# 🎓 AI-Powered University Academic Management Platform

## 📖 Overview

This project is an AI-powered system for managing university academic activities. It helps administrators, teachers, and students manage courses, attendance, results, and academic information efficiently.

---

## 🎯 Features

* Student Management
* Teacher Management
* Course Management
* Attendance Management
* Result Management
* Notice Management
* AI Academic Assistant
* Report Generation

---

## 👥 Users

### Admin

* Manage students
* Manage teachers
* Manage courses
* Publish notices

### Teacher

* Manage attendance
* Upload results

### Student

* View courses
* View attendance
* View results
* Use AI assistant

---

## 🛠️ Technology Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* AI: OpenAI API
